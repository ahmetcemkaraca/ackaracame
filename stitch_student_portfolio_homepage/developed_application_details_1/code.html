The Challenge

Traditional architectural planning involves manual, iterative processes for optimizing space utilization, which are often time-consuming and may not yield the most efficient layouts. The challenge was to create a tool that could automate and enhance this process, providing architects and designers with data-driven suggestions to maximize spatial efficiency and functionality while adhering to complex constraints.

The Solution & Features

This AI-powered web application serves as an intelligent assistant for spatial planning. By inputting a floor plan and a set of constraints (e.g., number of rooms, desired adjacencies, window access), the tool's generative algorithms propose a variety of optimized layouts. This not only accelerates the initial design phase but also uncovers novel solutions that might be missed in a manual workflow.

  • Automated Layout Generation: Upload a building shell and receive multiple layout options in seconds.
  • Constraint-Based Optimization: Define custom rules for room placement, dimensions, and relationships.
  • Real-Time Performance Analysis: Each generated layout is scored on key metrics like circulation efficiency and daylight exposure.
  • Interactive Visualization: Explore and modify the AI-generated plans in an intuitive 2D/3D interface.

Technical Stack

The application leverages a modern tech stack to deliver a robust and responsive user experience. The backend's generative logic is built with Python for its powerful data science libraries, while the frontend is developed with React for a dynamic and interactive user interface.
